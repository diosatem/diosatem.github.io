How's the Weather?

Website Planning Document

Site Name

This site is going to be called "HowstheWeather.com". The name clearly reflects the purpose of this site and it is easy to remember (since this is used by everyone when asking for weather conditions).

Site Purpose

The main purpose of this site is to inform people about the weather and second is to help them prepare for any type of weather disturbances that might impact their livelihood and outdoor activities. These purposes will be fulfilled through real-time advisories and warnings, community forums, forecasting services, and news releases.
